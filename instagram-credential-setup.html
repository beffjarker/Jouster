Step 6: Create and Connect Facebook Page

  1. Create a Facebook Page: Create Page

  3. Connect your Instagram to the Facebook Page

Step 7: Generate Page Access Token

Use the Graph API Explorer to generate your access token:

  1. Go to Graph API Explorer
  2. Select your app from the dropdown
  3. Select your Facebook Page (not "User")
  4. Click "Get Token" ‚Üí "Get Page Access Token"
  5. Select your page and grant permissions
  6. Add permission: instagram_basic
  7. Click "Generate Access Token"
Test your token: Make a GET request to /me to verify it works

Step 8: Get Instagram Business User ID

Use the Graph API Explorer with your page token:

  1. In Graph API Explorer, make sure you're using your Page Access Token
  2. Make a GET request to: /{your-page-id}?fields=instagram_business_account
  3. Copy the Instagram Business Account ID from the response
Test your setup: Make a GET request to /{instagram-user-id}/media?fields=id,media_url,permalink,caption
