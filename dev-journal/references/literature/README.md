# Literature Reference Collection

> **Purpose:** Public-domain works downloaded from [Project Gutenberg](https://www.gutenberg.org/), converted to Obsidian-compatible Markdown and split by chapter.  
> **Location:** `dev-journal/references/literature/`  
> **Tool:** `dev-tools/scripts/gutenberg-downloader.js`

---

## 📂 Folder Structure

```
literature/
  README.md                        ← This catalog
  _index.md                        ← Master Map of Content (MOC)
  {author-slug}/
    _index.md                      ← Author overview
    {title-slug}/
      _index.md                    ← Book overview with chapter links
      ch-01.md                     ← Chapter 1
      ch-02.md                     ← Chapter 2
      ...
```

## 🛠️ How to Download

Run the downloader script from the repo root or `dev-tools/` directory:

```bash
# Install dependencies (first time only)
cd dev-tools && npm install

# Download a single book
node scripts/gutenberg-downloader.js --id 11

# Download all works by an author
node scripts/gutenberg-downloader.js --author "lewis-carroll"

# Download the entire catalog
node scripts/gutenberg-downloader.js --all

# Preview what would be downloaded (no writes)
node scripts/gutenberg-downloader.js --all --dry-run

# List all catalog entries
node scripts/gutenberg-downloader.js --list

# Or use the npm shorthand from dev-tools/
npm run gutenberg -- --all
```

### Options

| Option | Description | Default |
|--------|-------------|---------|
| `--id <id>` | Download single book by Gutenberg ID | — |
| `--author <slug>` | Download all works for author slug | — |
| `--all` | Process entire catalog | — |
| `--list` | List catalog without downloading | — |
| `--output <path>` | Output directory | `dev-journal/references/literature` |
| `--dry-run` | Preview without writing | `false` |
| `--skip-existing` | Skip already-downloaded books | `false` |
| `--delay <ms>` | Politeness delay between requests | `2000` |
| `--verbose` | Detailed output | `false` |
